# Academic Records & Registrar System

A Java Swing application developed using Object-Oriented Programming (OOP) principles 
and the MVC (Model-View-Controller) architecture.

## Features

* Add Student Records
* Add Faculty Records
* Search Records by ID or Name
* Update Existing Records
* Delete Records
* Save Data to File
* Load Data from File
* Course Management for Students
* Input Validation and Error Handling

## Technologies Used

* Java
* Java Swing GUI
* MVC Architecture
  - Model
  - View
  - Controller
* Object-Oriented Programming (OOP)
  - Abstraction
  - Encapsulation
  - Inheritance
  - Polymorphism
* Java Collections Framework
  - HashMap
  - ArrayList
* File I/O using Java NIO
* Exception Handling

4. Run the Main class.
5. The Academic Records & Registrar System GUI will launch.
